## 1. 게임 개요

| 항목 | 내용 |
|---|---|
| 장르 | 방치형 육성 시뮬레이션 |
| 테마 | 스트리머(이베코짱)의 방송국을 매니저로서 운영 |
| 핵심 루프 | 방송 ON → 재화 획득 → 가챠(영업) → 아이템으로 케어 → 더 효율적인 방송 |
| 성장 축 | ① 구독자(채널 체급) ② 인연 레벨(호감도) |

## 2. 재화 및 스테이터스
**공통 규칙 - 정산 처리 규칙**
- 매 틱(초/분)마다 계산되는 수익에 소수점이 발생할 경우, 소수점을 버리지 않고 내부 버퍼(Buffer)에 누적 보관함.
- 누적된 버퍼 값이 1.0 이상(정수)이 되는 순간 실제 유저의 재화 총합에 가산되며, 화면(UI)에는 항상 정수 단위만 표기함.

### 2-1. 구독자 (Subscribers)

| 항목 | 설명 |
|---|---|
| 역할 | 타 게임의 "전투력"에 해당. |
| 성격 | **절대적 성장 지표 — 감소 없이 누적만 됨** |
| 증가 조건 | 방송 ON 상태일 때 시간에 따라 지속 증가 (방송 ON인 상태에서 게임 종료시 미접속 상태에서도 증가) |
| 감소 조건 | 없음 |
| 표기 규칙 | [6-4. 숫자 표기 규칙](#6-4-숫자-표기-규칙-formatting-rules) 참고 |

### 2-2. 방송 자금 (Broadcast Funds)

| 항목 | 설명 |
|---|---|
| 역할 | 소비형 재화. 매니저의 활동비 및 채널 유지비 개념 |
| 증가 조건 | 방송 ON 상태일 때 시간에 따라 지속 증가 (방송 ON인 상태에서 게임 종료시 미접속 상태에서도 증가) |
| 감소 조건 | 가챠(영업) 이용 시 소모 · 텐션 0일 때는 수익이 거의 0에 수렴 |

> **사용처 (확정 + 예정)**
> | 사용처 | 상태 | 비고 |
> |---|---|---|
> | 영업(뽑기) | ✅ 확정 | 1회 2,000 자금 / 10연 20,000 자금 |

> **사용처 (검토 중)**
> | 사용처 | 비고 |
> |---|---|
> | 단기 지속형 버프 구매 | 구독자/자금 획득량 증가 |
> | 영구 지속형 버프 스킬 구매 | 구독자/자금 획득량 증가|

### 2-3. 텐션 (Tension) — 최대 100

| 항목 | 설명 |
|---|---|
| 역할 | 스트리머의 기분·감정 곡선. 방송 수익 효율을 직접 결정 |
| 0이 되면 | 자동으로 방송 종료 |

**증가 수단:**

| 수단 | 회복량 | 제한 |
|---|---|---|
| 데일리 스타일링(세팅 메뉴에서 이용 가능) | 텐션 Max(100)까지 즉시 회복 | 하루 1회 |
| 음식 사용 | 아이템 등급별 상이 ([5-1. 아이템 카테고리] 참조) 일일 상한 200 | 가챠 획득 의존 |
| 소통(터치) | 성공/대성공 여부에 따라 일정량 회복 ([3-5. 소통] 참조) | 에너지 충전식, 일일 상한 200 (음식과 별도 계산) |
| 방송 OFF 휴식 | 1분당 +1 (1시간 40분에 0→100으로 완전 충전) | — |

**감소 요인:**

| 요인 | 상세 |
|---|---|
| 방송 피로 | 방송 ON 시 시간 경과에 따라 하락 (모드에 따라 하락 속도 다름) |

### 2-4. 인연 레벨
| 항목 | 설명 |
|---|---|
| 역할 | 스트리머와의 인연 레벨 표시 |
| 성격 | 얼마나 캐릭터와 교감했는지 나타내는 지표로, 절대 감소하지 않음 |
| 인연 경험치 획득 | 데일리 스타일링 진행시 100, 소통(터치)모드에서 유효 소통(텐션이 올라가는 소통)을 했을 때 획득한 텐션량만큼 경험치 증가 |
| 감소 조건 | 없음 |
| 표기 규칙 | 하트 속 숫자로 표기. 레벨은 1~999까지 |
| 경험치 테이블 | 미정 |

## 3. 시스템 설계
### 3-1. 방송 토글 (▶ LIVE)

| 상태/모드 | 컨셉 | 텐션 소모(3분당) | 자금 배율 | 구독자 배율 | 전략적 용도 |
|---|---|---|---|---|---|
| 소통 | [기본형] 안정적인 소통 | x1 | ×1.0 | ×1.0 | 평소 방치 시 기본 상태 |
| 게임 | [성장형] 구독자 펌핑 | x1.5 | ×0.7 | ×2.0 | 매니저 승급 조건 달성이 급할 때 |
| 노래 | [수익형] 후원금 집중 | ×2.0 | ×2.0 | ×0.5 | 가챠 자금이 급하게 필요할 때 |
| 눕방 | [휴식형]  | ×1 | ×0.1 | ×0.25 | 텐션이 1~29일 때 강제로 발동하는 모드 |

> 방송 ON 플로우
- ON 클릭 → 모드 선택 팝업 → 준비 화면(30초/터치 15초 단축)
- OFF 클릭 → 정산 화면 → 모드 선택 or 닫기
- 정산 화면에서 모드 전환 시 준비 15초(터치 7초 단축)

### 3-2. 영업 · 가챠 (Sales & Gacha)

방송 자금을 투입하여 매니저가 외부 협찬을 따오는 **'활동비 지출' 개념의 가챠**.

| 항목 | 내용 |
|---|---|
| 광고 보상 | 광고 시청 1회당 **무료 10연** · 일일 **1회** 기회 |
| 보상 | 의상(영구 자산) 또는 소비 아이템(텐션 회복용) |
| 상세 비용 및 확률 | [5-2. 가챠 경제 요약] 참조 |

> **천장(Pity) 시스템
> - 가챠 누적 **100회** 시 의상 아이템 하나 확정
> - 무료 가챠로는 누적되지 않음

**중복 의상 환전:**

| 등급 | 환전 재화 |
|---|---|
| Common | 100 재화 |
| Rare | 500 재화 |
| Legendary | 10000 재화 (가챠 5회분 리턴) |

### 3-3. 케어 (Care)
- 가챠로 획득한 **소비성 아이템(음식)**을 사용하는 전용 메뉴
- **초과분 폐기:** 텐션 100 상태에서도 사용 가능하나 100 초과분은 버려짐
- 음식으로 회복 가능한 하루 텐션은 최대 200 제한 (소통과 별도) 회복
- 회복 상한은 케어 메뉴 안쪽에 표시됨 (회복 가능 수치 N/200 형태)

### 3-4. 세팅 · 옷장 (Setting / Wardrobe)
- 가챠로 획득한 의상을 관리·교체하는 메뉴. 획득 시 바로 옷장에 추가됨.
- 헤드/바디 슬롯에 의상을 장착해 수익 배율을 높임 (구체적인 스탯은 [5-1. 아이템 카테고리] 참조)
- '데일리 스타일링' 시스템: 
  - 하루 **1회** 의상 교체 시 → **텐션 Max(100) 즉시 회복** + 인연 경험치 획득
  - 세팅 메뉴 안에 '데일리 스타일링 진행' 이라는 체크 박스가 있음
  - 의상을 단순 교체할 것인지 텐션 회복을 동시에 진행할 것인지 확인하는 체크박스. 디폴트는 체크 해제

### 3-5. 소통 · 터치 (Touch)
| 항목 | 내용 |
|---|---|
| 방식 | 에너지 충전식 |
| 에너지 충전 속도 | 2시간에 1개 충전 |
| 최대 보유량 | 5개 (단, 겉으로 명시되지 않는 내부 수치) |
| 에너지 사용법 | '소통' 메뉴를 눌러 진입한 다음 캐릭터 터치 |
| 에너지 1개당 회복 텐션량 | 20 또는 40 |
| 일일 텐션 회복 상한 | 합계 200 |
| 성공/실패 여부 | 반드시 성공만 있으며, 대성공 여부를 판정하여 20% 확률로 20 대신 40이 증가 (단, 상한선인 200을 넘어서 회복되지 않음) |
| 성공/대성공 이펙트 | 성공시 작은 하트가 1~2개 정도 캐릭터 주변에 나타났다가 사라지며, 대성공 시 성공보다 좀더 큰 사이즈의 하트가 4~5개 정도 캐릭터 주변에 나타났다가 사라짐|
| 과도한 터치 | 에너지가 0개인 상태에서 터치 시 텐션 미상승 / 지루해하는 반응 |

### 3-6. 매니저 레벨 & 호감도 (Manager Level)

**성장 이원화:**

| 축 | 올리는 방법 |
|---|---|---|
| **구독자 수** (채널 체급) | 방송 ON 시 자동 누적 (감소 없음) |
| **인연 레벨** (호감도) | 데일리 스타일링, 텐션이 오르는 유효 소통(터치) |

> 인연 레벨 상승시
- 가챠로 안 나오는 인연 레벨 전용 보상 의상 세트 제공

### 3-7. 개인 통합 메뉴 (Manager's Dashboard)

메인 화면의 **인연 레벨(하트 아이콘)** 클릭 시 노출되는 통합 관리 모달. 유저의 성장 지표와 데일리 과제, 그리고 데이터 분석을 3개의 탭으로 제공한다.

#### **[Tab 1] 커리어 (Career)**

매니저로서의 정체성과 채널의 장기적인 목표를 확인하는 탭.

| 항목 | 상세 내용 | 비고 |
| :--- | :--- | :--- |
| **인연 레벨** | 현재 인연 레벨 표시 (예: 인연 레벨 Lv.15) | - |
| **승급 조건** | 다음 등급까지 필요한 구독자 수 ($Target - Current$) | 진행도 바(Progress Bar) 병행 |
| **등급 전용 버프** | 등급마다 얻는 '영구 고정 버프' 수치 명시 | 예: 구독자 +15%, 자금 +10% |

#### **[Tab 2] 데일리 케어 (Daily)**

오늘의 숙제를 체크하는 탭.

| 항목 | 상세 내용 | 비고 |
| :--- | :--- | :--- |
| **일일 업무 현황** | 식사 (N/200), 데일리 스타일링(N/1), 소통 (N/200) | 총 3개 항목 |

#### **[Tab 3] 채널 분석 (Analysis)**

모든 버프가 합산된 '최종 효율'을 뜯어보는 전략 탭.

| 항목 | 상세 내용 | 비고 |
| :--- | :--- | :--- |
| **전체 버프 합산** | 채널 등급 보너스, 장착 의상 보너스 | 요인별 세부 리스트업 |
| **실시간 성장 지표** | 구독자 증가율: 약 +N명 / min, 약 +N원 / min | 배율은 기본 소통방송 모드를 반영 |

## 4. 밸런스 수치표

### 4-1. 스테이터스 증감 속도

텐션 최대치는 **100** 고정.

| 항목 | 상태 | 주기 | 초당 증감량 | 방전/완충 시간 |
|---|---|---|---|---|
| **텐션** | LIVE ON | 3분당 -1 (기본) |  |  |
| | LIVE OFF | 1분당 +1 | +1/60 | 약 1.6시간 (0→100) |

### 4-2. 방송 수익 엔진 (LIVE ON)

| 텐션 구간 | 상태 명칭 | 3분당 자금 | 3분당 구독자 | 비고 |
|---|---|---|---|---|
| 90 ~ 100 | 피버 모드 | 3000 | 5 | 알고리즘을 탄 최고 상태 |
| 70 ~ 89 | 하이 텐션 | 1000 | 3 | 최적의 관리 상태 |
| 30 ~ 69 | 보통 상태 | 500 | 2 | 일반적인 방송 |
| 1 ~ 29 | 피로 상태 | 30 | 0.5 | 방송이 강제로 '눕방' 으로 변경. 눕방 중에도 텐션은 기본 속도 (3분당 -1)로 하락 |
| 0 | 방송 종료 | 0 | 0 | 방송 강제 종료 |

### 4-3. 개발 상수 (코드 기준값)
// ── 스테이터스 감소/회복 ──
TENSION_DECAY_PER_SEC:   1 / 180,        // LIVE ON  : 3분당 −1
TENSION_REGEN_PER_SEC:   1 / 60,         // LIVE OFF : 1분당 +1

// ── 방송 수익 (3분당 → 초당 환산) ── 후원금 배율
EARNINGS_FEVER_PER_SEC:  3000 / 180,     // 텐션 90~100 (3분당 3,000원)
EARNINGS_HIGH_PER_SEC:   1000 / 180,     // 텐션 70~90  (3분당 1,000원)
EARNINGS_MID_PER_SEC:     500 / 180,     // 텐션 30~70  (3분당 500원)
EARNINGS_LOW_PER_SEC:      30 / 180,     // 텐션 0~30   (3분당 30원)

// ── 구독자 배율
SUBS_FEVER_PER_SEC:        5 / 180,      // 텐션 90~100
SUBS_HIGH_PER_SEC:         3 / 180,      // 텐션 70~90
SUBS_MID_PER_SEC:          2 / 180,      // 텐션 30~70
SUBS_LOW_PER_SEC:          1 / 180       // 텐션 0~30


## 5. 아이템 설계
### 5-1. 아이템 카테고리
| 타입 | 아이템명 | 주요 효과 (기본) | 등급별 보너스 수치 (Buff Value) | 비고 |
|:---:|:---:|:---|:---|:---|
| 영구 | 헤드 (Head) | 방송 자금 획득량 증가 | 일반 10% / 희귀 25% / 전설 50% | 곱연산 적용 |
| 영구 | 바디 (Body) | 구독자 증가량 증가 | 일반 10% / 희귀 25% / 전설 50% | 곱연산 적용 |
| 영구 | 세트 효과 | 텐션 하락 속도 감소 | 세트 완성 시 20% 감소 | 고정 수치 |
| 소비 | 음식 (FOOD) | 텐션 즉시 회복 | 일반 5 / 희귀 10 / 전설 30 | 일일 상한 200 |
| 소비 | 버프 (BUFF) | 각종 일시 버프 | 미구현 | 추후 업데이트 |

> 음식 등급별 회복량
|일반(COMMON)|5|
|희귀(RARE)|10|
|전설(LEGENDARY)|30|

### 5-2. 가챠 경제 요약

| 항목 | 수치 |
|---|---|
| 가챠 1회 비용 | 20,000 재화 |
| 10연차 비용 | 200,000 재화 |
| 일일 무료 (광고) | 10연 × 1회 |
| **일일 무료 합계** | **최대 10회**
| 의상 출현율 | 1.5% |
| 음식 출현율 | 98.5% |

### 5-3. 같은 카테고리 내 등급별 확률
| 항목 | 수치 |

## 6. UI / 연출 사양
### 6-1. 현실 시간 동기화 (오프라인 보상)
- 미접속 시에도 방송 상태에 따라 정산 처리
- 현실 시간에 따른 캐릭터/배경 색상 빛 표현 (배경 이미지 다원화, 캐릭터 필터 연출)

### 6-2. 매니저 종합 인포 대시보드 (3-Tab)
하트 모양의 매니저 레벨 아이콘 클릭 시 노출.

| Tab | 이름 | 내용 |
|---|---|---|
| 1 | **커리어** | 현재 등급, 레벨(XP), 다음 승급까지 필요 구독자 목표, 등급 전용 영구 버프 수치 |
| 2 | **데일리** | 오늘의 케어 현황(식사 0/200, 세팅 0/1 등) |
| 3 | **분석** | 현재 적용 중인 전체 버프 합산 리스트, 실시간 시간당 구독자/방송 자금 획득 지표 |

### 6-3. SSR 연출
- 등급이 높은 협찬(Legendary) 확정 시 매니저의 자신감 넘치는 **전용 대사** 출력

### 6-4. 숫자 표기 규칙 (Formatting Rules)

전광판 및 금액 출력 규칙.

| 조건 | 표기 방식 | 예시 |
|---|---|---|
| 7자리 이하 (0 ~ 9,999,999) | 3자리마다 콤마 | `1,234,567` |
| 8자리 이상 (10,000,000~) | 단위 축약 (M, B, T) | `10.5M` |

- $10^6$: **M** · $10^9$: **B** · $10^{12}$: **T**
- 소수점 첫째 자리까지 표시하되, `.0`이면 정수만 노출
